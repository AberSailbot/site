Yet Another Really Busy Day

Today, once again, we had a really productive day. We drilled all of the waterproof connector holes, and fitted all but 2 of the connectors because the delivery hasn't come yet. We added an LED to the power ditribution board so that we can see when it's on better. We also considered putting a waterproof switch outside of the boxes to make it easier to turn the power on and off. Louis tested and rewrote the GPS code to make it more effiecient and less buggy. We cabled the XBee box up to the main box.

Importantly we tested the sail winch and rudder servo with the RC remote, which showed we were getting an abnormal amount of interpherance in the connection. We diagnosed the problem as either radio interpherence from the room, power interpherance from the regulators/batteries, which could be fixed by some bigger resistors, or inteference down the cable attaching the servo's to the other boxes, which can be fixed by shortening the cable.
