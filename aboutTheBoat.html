Overview

Our entry this year is a 1.3 meter long custom built fibreglass boat. It uses a combination of an electronic compass, a GPS receiver and a wind sensor to detect its orientation, position and the wind direction. These are connected to a Raspberry Pi, which is a cheap single board ARM based computer, running Linux, and an Arduino Uno micro-controller board. The sail is controlled by an electronic winch and the rudder by a servo motor, both controlled by the Arduino.

Specifics

Software

The main logic is written in Java, running on the Raspberry Pi, which then communicates to the Arduino via some Python code, due to the fact that Java's serial communication libraries are horrible to use.

The Python code reads in from all the sensors via the Arduino, except the GPS, which is connected directly to the Pi. Then, when requested, it forwards them on to the Java code via a TCP/IP socket.

Another reason we decided to do this was because we wanted to leave the Java code able to be tested and developed using our boat simulator, which is a version of the open source TrackSail game which was edited to be able to be controlled via a TCP/IP socket.

The Python code will also communicate to our laptop, via an XBee (a long range wireless communication board), so that we can view the sensor readings and set/remove/edit way points from the shore, whilst the boat is still sailing.

The telemetry code on the computer is written in Java, and displays an OpenStreetMap of the area with the boats location indicated as well as the readings from all the sensors. It also allows the user to click on the map, to set a way point on the boat.

The Arduino code is written in the Arduino variant of C and simply reads from the appropriate sensor when it gets the command from the Pi. It also has a command to calibrate the wind sensor, to avoid the data it gives out losing accuracy over time.

Hardware

Of course there is the Raspberry Pi, the Arduino and the XBee as the control system, but other than that, we have 3 sensors and 2 servos:

  • We have a GPS, with an external antennae which is directly connected to the Raspberry Pi via Serial, and is mounted inside the boat with the electronics
  • We have a tilt-compensating compass, connected to the Arduino via Serial, which is mounted at the very front of the boat just under the deck
  • We have a home-made wind sensor, which is mounted on a pole at the back of the boat
  • As an actuator to move the sails there is a sail winch designed for RC boats mounted through the middle of the deck.
  • And as an actuator to move the rudder we also have a single waterproof servo mounted into the deck at the rear

Finally we have an RC receiver inside the boat which allows us to take control of the boat manually via an RC remote control if something goes wrong during the sail.
